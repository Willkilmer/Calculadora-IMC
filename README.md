# 🧮 Calculadora de IMC

Aplicação web desenvolvida para calcular o Índice de Massa Corporal (IMC) com base no peso e altura informados pelo usuário.

## 🚀 Tecnologias utilizadas
- HTML5
- CSS3
- JavaScript

## 📋 Funcionalidades
- Entrada de peso e altura
- Cálculo automático do IMC
- Exibição da classificação (Abaixo do peso, Normal, Sobrepeso, etc.)
- Interface simples e intuitiva

## 🎯 Objetivo do projeto
Praticar manipulação do DOM, captura de eventos e lógica de programação com JavaScript.
